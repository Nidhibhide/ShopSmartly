

# ShopSmartly - E-commerce Price Comparison Platform

## Overview

I designed and developed **ShopSmartly**, a dynamic e-commerce price comparison platform that empowers users to make informed purchase decisions. The platform enables users to search for products, compare prices across multiple e-commerce websites, and choose the best deals. It features a seamless user experience for login, signup, password recovery, and price comparison, with future plans to integrate more platforms. 

The project was built using the **MERN stack**, ensuring scalability and efficiency. This platform highlights my expertise in full-stack development and the integration of APIs to fetch and display real-time product data.

## Tools Used

- **MongoDB**
- **Express.js**
- **React.js**
- **Node.js**
- **REST APIs**
- **Material-UI**
- **Tailwind CSS**
- **JWT Authentication**

## Features

1. **User Authentication**
   - **Login/Signup:** Secure authentication and registration system.
   - **Forgot Password:** Recovery options for user accounts.
   - **JWT Authentication:** Ensures secure session management.

2. **Price Comparison**
   - **Search Products:** Intuitive search for desired products.
   - **Compare Prices:** Real-time price comparisons across three e-commerce platforms.
   - **Future Expansion:** Plans to integrate additional e-commerce websites.

3. **User-Friendly Design**
   - Responsive UI built with **React.js**, **Material-UI**, and **Tailwind CSS** for seamless navigation across devices.

## Admin Panel Responsibilities (Planned Features)

- Manage e-commerce platform integrations.
- Track user activities and preferences.
- Monitor and update product data APIs.
